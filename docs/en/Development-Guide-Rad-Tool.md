# ASP.NET Zero Power Tools

This legacy RAD Tool document has been replaced by the new ASP.NET Zero Power Tools documentation.

Power Tools is now installed as a .NET global tool and opens a web-based UI. Older RAD Tool workflows are no longer the recommended documentation path.

## Install and Run

Run the install command from the ASP.NET Zero `aspnet-core` folder so the project-provided `NuGet.Config` is used:

```bash
dotnet tool install -g AspNetZero.PowerTools
```

Then launch Power Tools:

```bash
aspnetzero-powertools
```
